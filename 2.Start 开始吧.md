# 一个简单的例子

```
<!DOCTYPE html>
<html> 
<head> 
    <link href="../legoland.min.css" rel="stylesheet">
</head> 
<body>
    <div class="lg-wrapper">
        <div class="lg-head">
            <div class="lg-nav">
                <div class="lg-nav-hd"><span class='lg-nav-title'>管理平台</span></div>
                <div class="lg-nav-by">
                    <ul> 
                        <li><a>一级菜单</a></li>
                        <li class="active"><a>一级菜单</a></li> 
                    </ul>
                </div> 
                <ul class="lg-nav-right">
                    <li><a>退出</a></li>
                    <li><span>管理员</span></li>
                </ul>
            </div>
        </div>
        <div class="lg-content">
            <div class="lg-content-left">
                <div class="lg-sidebar">
                    <ul class="lg-sidebar-l1"> 
                        <li>
                            <a class="active" href="javascript:;">
                                <span class="nav-label">二级菜单</span>
                            </a>
                            <ul class="lg-sidebar-l2"> 
                                <li>
                                    <a href="listpage.html">三级菜单</a>
                                </li> 
                            </ul>
                        </li>
                    </ul>
                    <div class="lg-sidebar-hand"></div>
                </div>
            </div>
            <div class="lg-content-right">
                <ul class="lg-breadcrumb">
                    <li><a>面包屑</a></li>
                    <li><a>首页</a></li>
                    <li><a>详情</a></li>
                </ul>
                <div class="lg-grid-container"> 
                    <div class="lg-grid-row">
                        <div class="lg-grid-12">
                            <div class="lg-body lg-body-full"> 
                            </div> 
                        </div> 
                    </div>
                </div>
            </div>
        </div>
</body>   
</html>
```
 

> - 任何一个管理后台的页面都可以从这个例子开始，你只需要正确地引入legoland.min.css文件

> - 通用的样式模块引用也非常简单，在模块示例文件中复制对应的 HTML 结构，就能像乐高玩具一样轻松搭建你需要的页面

> - 你需要做的只是在.lg-body块中搭建你的个性化页面，就是这么简单
